    PromptLink

    Project Summary:

    PromptLink is a full-stack AI prompt-routing platform that intelligently detects user intent and dynamically routes it to the most suitable large language model (LLM). It features a modern React-based frontend with a smooth, chat-style interface, a FastAPI backend integrated with LangChain for prompt classification and routing, and a Neo4j graph database for decision traceability. PromptLink includes semantic scoring, chain-of-thought validation, and enhancement logic to ensure responses are accurate and high quality. All interactions are cached, scored, and visualized for explainable AI delivery.

    Features:

    • LangChain-powered intent classification and dynamic LLM selection (GPT-4o, Gemini, GPT-3.5)
    • Custom scoring system using semantic similarity, length, fluency, and CoT validation
    • "Enhance Response" feature for regenerating low-quality replies with improved instructions
    • Firebase authentication with session persistence and real-time error handling
    • Context-aware caching to minimize latency and API calls
    • Neo4j graph logging to visualize prompt → intent → model → response → score
    • Responsive React + Vite frontend with Markdown rendering and code highlighting
    • Popovers that explain the model’s reasoning and routing logic on hover

    Technologies Used:

    Python, JavaScript, FastAPI, React, Vite, LangChain, Neo4j, Firebase Auth, Prism.js, Numpy, Radix UI

    InventoryHub

    Project Summary:

    InventoryHub is a robust inventory management system designed to empower small businesses and hobbyists to efficiently track, manage, and optimize their stock. The platform features a modern Flask-based web interface with a responsive design, persistent SQLite storage for data reliability, and seamless in-memory caching powered by custom C data structures (hash map and B-tree) integrated via Python’s ctypes. InventoryHub offers secure user authentication and role-based access control, ensuring that inventory operations are both efficient and secure.

    Features:

    • Secure user authentication and role-based access control for managing inventory
    • Responsive dashboard displaying detailed item information, including the username of the user who added the item
    • Full CRUD functionality (add, update, delete) for inventory items with permission-based access
    • In-memory caching using a custom C-based hash map for fast, constant-time data retrieval
    • Docker containerization, CI/CD pipelines, and hosting on Render ensure reliability and growth potential.
    • Efficient sorted search and range queries enabled by a custom B-tree implementation

    Technologies Used:

    Python, C, Flask, Docker, Bootstrap, SQLite, ctypes, Gunicorn
