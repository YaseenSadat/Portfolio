    Lumenera

    Project Summary:

    Lumenera is a full-stack dark fantasy trading card platform designed for collectors and players, offering a seamless and secure experience with features like card browsing, account management, and dynamic purchase systems. Built with the MERN stack, it integrates modern tools and technologies to ensure scalability, reliability, and user satisfaction.

    Features:

    • Account Management: Secure login, signup, and logout with JWT authentication and bcrypt password hashing.
    • Password Reset: Encrypted, email-based reset system with time-sensitive tokens.
    • Home Page: Responsive and intuitive design showcasing products, cart, and user management features.
    • Product Browsing: Detailed descriptions, advanced search, filters, and sorting options.
    • Secure Checkout: Stripe integration for safe and efficient payment processing.
    • Email Notifications: Automated emails for subscriptions, purchases, and password resets using Nodemailer.
    • Admin Dashboard: Comprehensive tools for managing products, orders, and platform settings with enhanced security.
    • Scalability and Security: Docker containerization, CI/CD pipelines, and hosting on Render ensure reliability and growth potential.
    • Craftsmanship: Unique, hand-crafted card designs with a rich dark fantasy aesthetic.

    Technologies Used:

    • Languages: JavaScript
    • Frameworks: React, Node.js, Express.js, Tailwind CSS, Jest
    • Libraries: react-router-dom, axios, react-toastify
    • Tools: Docker, Jenkins, JWT, Stripe, Nodemailer, bcrypt, Cloudinary, Postman, multer, Nodemon, Cors, Babel, validator
    • Database: MongoDB
    • Hosting: Render

    StockFlow

    Project Summary:

    StockFlow is an interactive stock trading simulation app designed to provide a risk-free and educational trading experience. Built with clean architecture principles, it empowers users to explore stocks, analyze data, and simulate trades with an AI-powered chatbot offering personalized financial guidance.

    Features:

    • Account Management: Secure login, signup, and logout functionalities for a seamless user experience.
    • Home Page: Centralized navigation hub with an intuitive layout for exploring features and managing stocks.
    • AI Chatbot: Personalized financial assistant to answer stock-related questions and explain technical indicators.
    • Explore Page: Access over 50,000 stocks, view historical and real-time data, and compare key indicators like EMA, SMA, and RSI.
    • Portfolio Management: Buy and sell stocks, view transaction history, and track performance through a personalized dashboard.

    Technologies Used:

    Java, Firebase, Java Swing, Polygon API, OpenAI API, Clean Architecture Principles.
